This folder defines the initial product and technical blueprint for a golf swing analysis platform focused on accurate feedback from fixed-angle smartphone video.

## North Star

Deliver trustworthy swing feedback by treating the system as a measurement pipeline first and an AI coach second.

## Scope for V1

- One-angle swing upload and playback.
- Frame scrubbing and slow-motion viewing.
- Manual and automatic line/circle overlays.
- Checkpoint-based feedback with confidence and explanation.
- Retake-required flow when capture quality is too low.
